StepLab 

Project Overview 

StepLab is a web application where users can browse, buy, review, and sell shoes. The platform is designed for sneaker enthusiasts and casual shoppers who want to discover the latest footwear, track their favorite brands, and find great deals on both new and used releases. 

Purpose 

Many users find it hard to get the newest shoe releases at an affordable price. StepLab provides a single place where users can explore both new and old shoe releases from all brands helping them save time and money while staying up to date with the latest trends. 

Intended Users 

Sneaker fans, collectors, and everyday shoppers looking to buy or sell shoes, discover deals, or stay updated on new sneaker drops 

Core Features 

Browse shoes by brand, category, or price range 

View detailed shoe information (images, prices, descriptions, sizes) 

Add or remove shoes from a favorites list or cart 

User login/signup and profile management 

Post shoes for sale and write reviews 

Fetch and display live sneaker data or news using an external API 

Responsive design for mobile and desktop 

Technologies & Tools 

Frontend: React  

Backend: Node.js + Express 

Database: PostgreSQL 

External API: Sneaks API (https://github.com/druv5319/Sneaks-API)
